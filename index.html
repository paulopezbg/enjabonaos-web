Jabones artesanales

JABÓN DE YOGURT GRIEGO Y ROSAS

Descongestionante, desinflamatorio, suavizante y difumina arrugas y líneas de expresión. Recomendado para piel normal y asfixiada.

JABÓN MENTA

Recomendado para pies cansados, adoloridos e hinchados. No recomendado para piel delicada o con irritaciones.

JABÓN DE YOGURT DE FRESA CON CHOCOLATE

Hidratante, suavizante, aclarante y bactericida. Aporta humedad y proteínas. Recomendado para piel normal y seca.

JABÓN MOCACCINO

Exfoliante, corta la grasa, aclarante, hidratante y suavizante. Recomendado para piel grasa, mixta y con acné.

JABÓN DE PEPINO Y LECHE

Para piel normal a sensible, especialmente para piel que se enrojece o inflama con el sol.

JABÓN PARA MANOS

Jabón de uso diario capaz de eliminar bacterias y virus. Recomendado para todas las edades.

JABÓN DE ARCILLA VERDE

Para piel grasa y mixta con acné o seborrea. Puede usarse en rostro, cuerpo y cuero cabelludo.

JABÓN DE CÚRCUMA

Para manchas en la piel, vitíligo, psoriasis y pieles opacas.

JABÓN CON DECORACIÓN PARA NIÑOS

Jabón con figuras encapsuladas. Para baño y uso diario en lavado de manos.

JABÓN DE ARROZ Y PEREJIL

Antienvejecimiento. Recomendado para piel madura (40–70 años) y también para piel grasa o mixta.

JABÓN ATARDECER (PARA CABELLO)

Para caída de cabello o cabello delgado; ayuda a evitar la caída y a engrosar el tallo capilar.

JABÓN NEUTRO

Jabón suave y sin fragancia, ideal para piel sensible y uso diario.

JABÓN DE VINO TINTO Y CALÉNDULA

Corta grasa, purifica, cicatriza, aclara, tonifica y desinflama la piel.

JABÓN LIMÓN–COCO

Para piel grasa a mixta pero sensible. También para cabello con caída parcial o total (alopecia, areata o por quimioterapia).
